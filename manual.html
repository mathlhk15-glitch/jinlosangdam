01 · 시스템 소개

이 프로그램은 무엇인가요?

진로진학상담교사가 학생 상담 전체 과정(예약 → 준비 → 기록 → 이력 관리)을 하나의 화면에서 처리할 수 있도록 만든 올인원 진로상담 관리 시스템입니다. 별도 서버 없이 HTML 파일 하나로 동작하며, 인터넷 없이도 사용 가능합니다.

🎯 핵심 특징 요약
설치 불필요 — 파일을 더블클릭하거나 학교 서버에 올리면 즉시 사용
완전 오프라인 — 인터넷 없이도 100% 작동 (데이터는 브라우저 내부에 저장)
학생 데이터 통합 — 상담 기록, 진로검사 결과, 과목 설계를 한 학생 아래 모두 연결
내장 과목 데이터베이스 — 2028학년도 고교학점제 과목 정보 및 대학별 권장과목 내장
🏠
대시보드
오늘 상담 일정, 전체 통계, 최근 기록을 한눈에 확인
📅
캘린더 · 예약
월간 달력으로 상담 예약을 시각적으로 관리
💬

학생 정보 조회, 기록 작성, 자동 요약, 체크리스트 설정
📋
상담 이력
누적된 상담 기록 검색·열람·인쇄
🔍
진로검사 결과
홀랜드 흥미·적성·가치관·성숙도 검사 결과 입력 및 대시보드 분석
📚
선택과목 설계
학년별 과목 선택 현황 기록 및 대학 권장과목 비교
🔗
진로정보 사이트
커리어넷, 워크넷 등 주요 진로정보 사이트 즐겨찾기

02 · 시작하기

처음 사용 — 5분 셋업

HTML 파일을 열면 바로 사용할 수 있습니다. 별도 설치나 로그인이 필요 없습니다.

1
파일 열기
진로상담시스템.html 파일을 더블클릭합니다.
Chrome 또는 Edge 브라우저에서 열리면 가장 안정적으로 동작합니다.
학교 서버(구글 사이트, GitHub Pages 등)에 올리면 URL로 접속 가능합니다.
💡
Edge나 Chrome을 추천합니다. Safari·Firefox에서도 동작하지만 일부 기능 차이가 있을 수 있습니다.
2
대시보드 확인
파일을 열면 즉시 대시보드 화면이 나타납니다. 왼쪽 사이드바에 전체 메뉴가 있습니다.
처음에는 데이터가 없으므로 통계가 모두 0으로 표시됩니다. 정상입니다.
3
학생 등록 (첫 번째 할 일)
왼쪽 메뉴에서 👥 학생 관리를 클릭 → ➕ 학생 추가 버튼으로 담당 학생들을 등록합니다.
이름, 학년, 반, 번호, 희망 진로, 희망 대학, 메모를 입력할 수 있습니다.
📌
학생을 먼저 등록해야 상담 예약과 진로검사 입력이 가능합니다. 가장 먼저 해야 할 작업입니다.
4
데이터 저장 방식 이해
모든 데이터는 브라우저의 localStorage에 자동 저장됩니다.
같은 컴퓨터·같은 브라우저에서 파일을 다시 열면 이전 데이터가 그대로 유지됩니다.
다른 컴퓨터로 이동하거나 브라우저 캐시를 지우면 데이터가 사라집니다 → 반드시 💾 데이터 백업을 정기적으로 실행하세요.
⚠️
중요: 브라우저 '방문 기록 삭제' 또는 'localStorage 초기화'를 하면 모든 상담 데이터가 삭제됩니다. 반드시 주기적으로 백업하세요.
03 · 메뉴 안내

전체 메뉴 구조

왼쪽 사이드바에 모든 메뉴가 있습니다. 클릭하면 해당 화면으로 이동합니다. 모바일에서는 상단 ☰ 버튼을 눌러 메뉴를 펼칩니다.

🏠
대시보드 메인
오늘 일정, 전체 통계, 미완료 과제, 최근 상담 기록을 한 화면에서 확인
📅
캘린더 · 예약 메인
월간 달력 + 당일 상담 목록. 상담 예약 추가/수정/취소
💬
상담 진행 상담
실시간 상담 기록 작성, 학생 정보 참조, 자동 요약 생성
📋
상담 이력 상담
학생별·날짜별 상담 기록 검색, 열람, 인쇄, 삭제
📚
선택과목 설계 학업설계
학년별 선택과목 기록 + 대학별 권장과목 비교 분석
🔍
진로검사 결과 학업설계
커리어넷·학교검사 결과 입력, 통합 분석 대시보드 확인
🔗
진로정보 사이트 진로정보
커리어넷, 워크넷, 대입정보포털 등 주요 사이트 바로가기
🏆
진로 월드컵 진로정보
직업을 토너먼트로 비교하는 학생 활동 도구
👥
학생 관리 관리
학생 등록·수정·삭제, 학생별 상담 이동
⚙️
설정 관리
교사 이름·학교명 설정, 데이터 백업·복원, 초기화

05 · 학생 관리

학생 등록 및 관리

모든 상담의 기반이 되는 학생 데이터를 관리합니다. 처음 사용 시 가장 먼저 해야 할 작업입니다.

1
학생 추가
메뉴 → 👥 학생 관리 클릭 → 우측 상단 ➕ 학생 추가 버튼 클릭
입력 항목: 이름(필수), 학년, 반, 번호, 희망 진로, 희망 대학, 메모
저장 버튼을 누르면 목록에 학생이 추가됩니다.
2
학생 수정 / 삭제
학생 목록에서 이름 옆 ✏️ 수정 버튼을 눌러 정보를 변경할 수 있습니다.
🗑️ 삭제는 학생 정보만 삭제되며, 상담 기록은 별도로 유지됩니다.
3
학생 검색 및 필터
상단 검색창에 이름을 입력하면 즉시 필터링됩니다.
학년 탭(전체/1학년/2학년/3학년)으로 학년별 목록을 볼 수 있습니다.
4
학생 카드에서 바로 이동
학생 카드의 💬 상담하기 → 해당 학생으로 즉시 상담 화면 이동
📊 진로검사 → 해당 학생의 진로검사 대시보드로 이동
💡
담당 학생이 많을 경우, 학번(학년+반+번호) 형식으로 메모를 남겨두면 URL 파라미터로 특정 학생을 바로 불러올 수 있습니다. (예: ?studentId=10315)
06 · 상담 진행

상담 진행 — 핵심 기능

실제 상담 중에 사용하는 화면입니다. 학생 정보와 과거 기록을 참조하면서 새 상담 내용을 기록합니다.

📋 상담 진행의 흐름 (STEP 구조)

이 시스템은 7단계 구조화 상담 프로세스를 지원합니다. 각 단계는 왼쪽 패널에서 순서대로 확인할 수 있습니다.

STEP 1
학생 선택
→
STEP 2
사전 정보 확인
→
STEP 3
진로 준비도 분석
→
STEP 4
상담 기록 작성
→
STEP 5
자동 요약
→
STEP 6
저장 확정
→
STEP 7
체크리스트 설정
1
상담 진행 화면 진입 방법
방법 A: 대시보드에서 오늘 일정의 학생 이름 클릭
방법 B: 학생 관리에서 학생 카드의 "💬 상담하기" 클릭
방법 C: 💬 상담 진행 메뉴 → 드롭다운에서 학생 선택 → "상담 시작"
2
왼쪽 패널 — 학생 정보 확인
상담 화면 왼쪽에 학생 정보 요약이 표시됩니다:
· 기본 정보: 이름, 학년/반/번호, 희망 진로, 희망 대학
· 진로 준비도 분석: 검사 결과 요약, 상담 전략 유형, 미완료 과제
· 성취수준: 과목별 성취수준(상/중/하) 시각화
· 상담 이력: 이전 상담 기록 타임라인
3
오른쪽 — 상담 기록 작성
텍스트 에디터에 상담 내용을 자유롭게 입력합니다.
상단 툴바로 굵게, 기울임, 구분선, 목록 등 간단한 서식을 적용할 수 있습니다.
학생이 사전에 남긴 질문/메모가 있으면 파란 박스로 상단에 표시됩니다.
4
🤖 자동 요약 생성 (선택)
기록 작성 후 "🤖 자동 요약 생성" 버튼을 누르면 상담 내용의 핵심이 자동으로 요약됩니다.
요약은 기록 저장 시 함께 포함되어, 나중에 이력을 빠르게 검토할 때 유용합니다.
💡
자동 요약은 오프라인에서도 동작하는 내부 규칙 기반 요약입니다. 외부 AI API를 사용하지 않습니다.
5
🚩 위기 상황 플래그
학교폭력·자해·자살 위험 등 위기 상황 감지 시 🚩 플래그 버튼을 누릅니다.
위기 유형(학교폭력/자해·자살 위험/기타)을 선택하고 저장하면, 해당 기록은 자동 삭제 대상에서 영구 제외됩니다.
⚠️
플래그 기록은 학교 보안 규정에 따라 별도 보관 절차를 따르세요. 이 시스템은 기록 보존만 지원하며 신고 절차는 별도로 진행하셔야 합니다.
6
💾 기록 저장
"💾 기록 저장" 버튼 클릭 → 내용 미리보기 팝업 확인 → "✓ 저장 확정" 클릭
저장과 동시에 해당 예약 세션의 상태가 '완료'로 변경됩니다.
7
📋 다음 상담 전 과제 설정 (체크리스트)
저장 직후 체크리스트 설정 팝업이 자동으로 열립니다.
기본 제공 항목(관심 학과 조사, 대학별 권장과목 비교 등) 중 선택하거나, 직접 입력해서 추가할 수 있습니다.
설정된 과제는 대시보드 '미완료 체크리스트'에 표시되어 다음 상담 시 확인을 유도합니다.
✅
체크리스트는 상담 연속성을 높이는 핵심 기능입니다. 과제를 설정하면 학생이 다음 상담까지 무엇을 해야 하는지 명확해집니다.
07 · 상담 이력

상담 이력 열람 및 관리

저장된 모든 상담 기록을 검색하고 열람합니다. 생기부 작성이나 상담 연속성 확인 시 자주 사용합니다.

기능사용 방법활용 예시
학생별 검색 상단 검색창에 학생 이름 입력 특정 학생의 전체 상담 이력 확인
기록 열람 목록에서 기록 클릭 → 상세 팝업 이전 상담 내용, 요약, 날짜 확인
인쇄 / PDF 저장 기록 팝업의 "🖨️ 인쇄·PDF" 버튼 학부모 면담 자료, 학생 제출 서류 준비
기록 삭제 목록의 🗑️ 버튼 클릭 (확인 필요) 잘못 저장된 테스트 기록 정리
위기 기록 식별 🚩 빨간 뱃지로 표시된 기록 확인 위기 상황 학생 모니터링
체크리스트 확인 기록 상세에서 체크리스트 항목 표시 과제 완료 여부 다음 상담 전 확인
08 · 진로검사 결과

진로검사 결과 입력 및 분석

커리어넷 및 학교 자체 검사 결과를 입력하면 통합 분석 대시보드에서 학생의 진로 프로파일을 시각적으로 확인할 수 있습니다.

🔍 지원하는 검사 종류
커리어넷 (온라인 검사) — 직업흥미검사, 직업적성검사, 직업가치관검사
학교 자체 (종이 검사) — 진로흥미유형(홀랜드 코드), 직업성격유형, 직업적성, 진로성숙도 포함 종합 검사
1
검사 결과 입력 방법
메뉴 → 🔍 진로검사 결과 → 우측 상단 "✏️ 결과 수동 입력" 버튼 클릭
팝업에서 학생을 선택(또는 신규 등록)하고 탭을 전환하며 각 검사 결과를 입력합니다.
탭 구성: 학교검사 종합 / 커리어넷 직업흥미 / 커리어넷 직업적성 / 커리어넷 직업가치관
2
학교검사 탭 입력 항목
· 흥미 유형: 홀랜드 코드 (예: 탐구형(I)-사회형(S)) — 1순위, 2순위 선택
· 직업 성격 유형: MBTI 스타일 4자리 코드 및 수준(상/중/하)
· 직업 적성: 추천직업 1·2, 수리력·공간력·추리력 등 개별 적성 항목
· 진로 성숙도: 진로계획성, 독립성, 진로탐색행동 수준(상/중/하)
· 희망직업: 학생의 현재 희망 직업
· 학생 소감: 검사 결과에 대한 학생 본인 의견
3
진로검사 대시보드 확인
결과 입력 후 학생 카드의 "📊 대시보드" 버튼 클릭 → 통합 분석 화면 이동
대시보드 구성: 검사 커버리지, 홀랜드 육각형, 강점 적성, 가치관 태그, 추천직업 및 반응 설정
4
추천직업 반응 설정
대시보드에서 추천직업 옆 [수용 / 보류 / 제외] 버튼을 눌러 학생의 반응을 기록합니다.
다음 상담 준비 시 AI 분석 포인트에 자동 반영되어 "추천직업 보류 이유 탐색 필요" 등의 알림이 생성됩니다.
📌
모든 항목을 입력하지 않아도 됩니다. 가진 정보만 입력해도 대시보드에서 커버리지 분석이 자동으로 이루어집니다. 미입력 항목은 "검사 미실시 → 상담 시 탐색 질문 제안"으로 표시됩니다.
09 · 선택과목 설계

고교학점제 과목 설계 지원

2028학년도 고교학점제 기반으로 학생의 학년별 선택과목을 기록하고, 목표 대학의 권장과목과 비교합니다.

1
학생 선택 및 학년 탭 이동
메뉴 → 📚 선택과목 설계 클릭
상단에서 학생을 선택하고, 1학년 / 2학년 / 3학년 탭으로 이동합니다.
각 학년에서 선택한 과목을 기록할 수 있습니다.
2
과목 클릭으로 정보 확인
과목명을 클릭하면 팝업으로 과목 상세 정보가 표시됩니다:
· 과목 유형(공통/일반선택/진로선택/전문교과)
· 연계 진로 및 대학 권장 여부
· 수능 선택과목 해당 여부
· 교사 안내 팁 및 주의사항
3
대학별 권장과목 비교
오른쪽 패널에서 희망 대학의 권장과목 데이터가 표시됩니다.
학생이 선택한 과목과 대학 권장과목이 일치하면 ✅, 누락이면 ⚠️로 표시됩니다.
내장된 2028학년도 권역별 대학별 권장과목 데이터가 기본 제공됩니다.
📌
내장 데이터 외에 추가 엑셀 파일(.xlsx)을 업로드하여 과목 데이터를 확장할 수 있습니다. 설정 메뉴에서 업로드하세요.

10 · 캘린더 · 예약 관리

상담 일정 관리

월간 달력에서 상담 예약 현황을 파악하고 새 예약을 추가합니다.

1
상담 예약 추가
메뉴 → 📅 캘린더·예약 클릭
우측 상단 📅 예약 추가 버튼 또는 달력의 날짜 클릭
입력 항목: 학생 선택, 상담 유형, 날짜, 시간, 사전 메모
2
상담 유형 선택
진로 상담 / 과목 선택 상담 / 진학 전략 상담 / 학부모 상담 중 선택
유형에 따라 달력 색상 점이 다르게 표시됩니다 (파랑/초록/주황/빨강).
3
당일 일정 확인
달력에서 날짜를 클릭하면 오른쪽 패널에 해당 날의 상담 목록이 표시됩니다.
목록의 상담 항목을 클릭하면 바로 상담 진행 화면으로 이동합니다.
💡
달력에 표시된 색상 점(dot)의 의미: 파랑=진로상담, 초록=완료, 빨강=위기 플래그. 점이 많은 날을 한눈에 파악할 수 있습니다.

12 · 데이터 관리

백업 · 복원 · 초기화

가장 중요한 데이터 보호 방법입니다. 반드시 숙지하세요.

⚠️
핵심 주의사항: 모든 데이터는 브라우저 localStorage에 저장됩니다. 브라우저 캐시 삭제, 개인정보 모드 사용, 다른 브라우저 사용 시 데이터가 보이지 않거나 삭제될 수 있습니다. 정기적인 백업이 필수입니다.
1
💾 데이터 백업 (내보내기)
화면 상단 오른쪽 "💾 데이터 백업" 버튼 클릭
→ JSON 파일이 자동으로 다운로드됩니다.
→ 파일명 형식: 진로상담_백업_2026-05-09.json
추천 주기: 매주 금요일 퇴근 전, 또는 상담 10건 진행 후마다
2
📂 데이터 불러오기 (가져오기)
화면 상단 오른쪽 "📂 불러오기" 버튼 클릭
→ 이전에 백업한 JSON 파일 선택
→ 확인 후 데이터가 복원됩니다.
주의: 불러오기 시 현재 브라우저의 기존 데이터가 덮어씌워집니다.
3
🖥️ 다른 컴퓨터로 이동 시
1. 현재 컴퓨터에서 데이터 백업 → JSON 파일 저장
2. HTML 파일과 JSON 파일을 USB 또는 이메일로 새 컴퓨터에 전달
3. 새 컴퓨터에서 HTML 파일 열기 → 데이터 불러오기 → JSON 파일 선택
4
⚠️ 데이터 초기화
설정 메뉴 → "데이터 초기화" 버튼 → 확인
모든 학생·상담 기록·검사 결과가 삭제됩니다. 반드시 백업 후 진행하세요.
학년 초 새로운 학생 명단으로 시작할 때 사용합니다.
⚠️
초기화는 되돌릴 수 없습니다. 반드시 백업을 먼저 하세요.
13 · 자주 묻는 질문

FAQ — 자주 묻는 질문

초보 선생님들이 가장 많이 막히는 상황을 정리했습니다.

Q
파일을 열었는데 이전 데이터가 없어요.
같은 컴퓨터, 같은 브라우저에서 열어야 합니다. Chrome에서 저장한 데이터는 Edge에서 보이지 않습니다.
브라우저 캐시를 삭제했거나 개인정보 모드(시크릿 모드)에서 열었다면 데이터가 사라진 것입니다.
→ 백업 파일(.json)이 있다면 📂 불러오기로 복원하세요.
Q
학생이 60명인데 일일이 입력해야 하나요?
현재 버전에서는 개별 입력이 기본입니다. 다만 이름, 학년, 반, 번호만 입력해도 기본 상담이 가능합니다.
희망 진로·대학은 상담 과정에서 파악한 후 추후 업데이트할 수 있습니다.
우선 상담이 필요한 학생부터 등록하고 점진적으로 확장하는 방식을 권장합니다.
Q
진로검사를 모두 실시하지 않은 학생도 입력할 수 있나요?
네. 가진 검사 결과만 입력하면 됩니다. 입력되지 않은 검사는 '검사 미실시'로 표시되며,
상담 준비 분석에서 "해당 검사 미실시 → 상담 중 탐색 질문 예시"가 자동으로 제안됩니다.
Q
학교 서버(구글 사이트 등)에 올려서 사용할 수 있나요?
네. HTML 파일 하나만 서버에 올리면 URL로 접속할 수 있습니다. 단, 파일을 올린 서버에서 외부 폰트·라이브러리를 인터넷으로 불러오므로 학교 방화벽 설정을 확인하세요 (googleapis.com, cdnjs.cloudflare.com 허용 필요).
데이터는 각 사용자의 브라우저에 저장되므로 서버 공유 시 데이터는 공유되지 않습니다.
Q
상담 기록을 인쇄하거나 PDF로 저장하려면?
상담 이력에서 기록을 클릭 → 상세 팝업의 "🖨️ 인쇄·PDF" 버튼 클릭
브라우저 인쇄 창이 열리면 대상: PDF로 저장을 선택하면 PDF 파일로 저장됩니다.
Q
모바일(스마트폰)에서도 사용할 수 있나요?
네. 모바일 반응형으로 설계되어 있습니다. 상단 ☰ 햄버거 버튼을 눌러 메뉴를 열 수 있습니다.
다만 긴 상담 기록 작성은 PC에서 하는 것이 훨씬 편리합니다. 모바일은 간단한 확인·조회에 활용을 권장합니다.
Q
다른 동료 교사와 데이터를 공유하려면?
백업 파일(.json)을 공유하면 됩니다. 받는 쪽에서 📂 불러오기로 데이터를 가져올 수 있습니다.
단, 불러오면 기존 데이터를 덮어씁니다. 중요한 데이터가 있다면 미리 백업하세요.
동시 접속 협업 기능은 현재 버전에서는 지원되지 않습니다.

✅ 처음 사용 체크리스트

□ 👥 학생 관리에서 담당 학생 5명 이상 등록했다
□ 📅 캘린더에서 이번 주 상담 예약을 추가했다

□ 💾 데이터 백업을 처음으로 실행해서 JSON 파일을 저장했다
□ 📱 스마트폰에서도 접속해보았다 (선택)
